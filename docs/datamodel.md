# Introduction

This document describes the snapsnare datastore.

# Tables

The following tables are recognized:
* access_modifiers (amr)
* activities (acy)
* application_properties (cpt)
* components (cpt)
* files (fle)
* fragment_assignments (fat)  
* fragments (fmt)
* genres (gre)
* icecast_statuses (iss)
* instruments (ist)
* jammers (jmr)
* policies (ply)
* principles (pce)
* registrations (rgn)
* replies (rpy)
* resets (rst)
* sections (stn)
* snaps (snp)
* templates (tpe)
* users (usr)
